# ⚡Mahmoud Widgets

Custom widgets and components ready to use under your awesome projects!

## 🎖Installing

```yaml
dependencies:
  mahmoud_widgets: ^<latest_version>
```

## ⚽Widgets

## 1. Rating 🔥

```
MahmoudWidgets.ratingWidget(
  color: Colors.red,
  size: 50,
  initialRating: 2,
  onRated: (int? value) {},
 ),
```
